README Chat Application

Description
	-this chat application project is an example of how a client and server send and recieve messages to each other

Motivation
	-the motivation for this project is learning about how a client and server chat


Getting Started
	- to get started run server.py then run client.py 
	- the server will prompt you to enter the Ip address and port number (hitting enter will create default ip and port)
	- change to client. 
	- the client will then prompt you with the same questions. Enter the same ip address and port number.
	- at this point you will be able to send messages between server and client.
